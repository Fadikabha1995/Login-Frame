This Login-in Frame starts with empty database and it has two labels in it
one to write the username and the second to write the password and then
you need to click on "Login" button.

*you can register a lot of members on it using "Create user" button.

*once you have some data you can view it all by clicking the "Show all users" button
 and it will open another frame showing all the users that you have been create
 and the passwords for each one

*if you try to login in using an exist user with a wrong password 3 time
 the user will locked out.

*you can unlock the user by clicking on "Unlock user" button but you need
 first to write the right credincials in Username and password labels
